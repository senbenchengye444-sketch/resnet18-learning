# data/

このディレクトリは、学習/評価データセットを格納するために使用されます。

## ⚠️ 注意：データセットの自動ダウンロード

データセットのファイル（CIFAR-10）は、容量が大きいため本リポジトリには含めていません。

`train.py` を実行する際、`torchvision.datasets.CIFAR10` の `download=True` オプションにより、データは自動的に `./data` ディレクトリ内にダウンロード・展開されます。

### 自動ダウンロードされるデータの内容

このディレクトリには、以下の内容を持つCIFAR-10データセットがダウンロードされます。

* **データ形式:** 32x32ピクセルのカラー画像（RGB形式）。
* **クラス数:** 10種類（飛行機、自動車、鳥、猫、鹿、犬、カエル、馬、船、トラック）。
* **総画像数:** 60,000枚。
    * 学習用データ（Train set）: 50,000枚
    * 評価用データ（Test set）: 10,000枚
